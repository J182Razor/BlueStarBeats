# Usage Guide - Blue Star Beats

Complete guide to using Blue Star Beats for brainwave entrainment.

## Getting Started

### First Time Setup

1. **Open the Application**
   - Navigate to `http://localhost:5173/` (development) or your deployed URL
   - The onboarding screen will appear

2. **Select Your Goal**
   - Choose from four primary goals:
     - 😴 **Sleep Better** - Fall asleep faster and achieve deeper sleep
     - 😌 **Reduce Stress & Anxiety** - Find calm and peace in minutes
     - 🧠 **Improve Focus** - Enhance concentration and enter flow states
     - 🧘 **Deepen My Meditation** - Achieve deeper meditative states

3. **Browse Sessions**
   - Click "📚 Session Library" to view available sessions
   - Filter by goal or view all sessions
   - Free users see 7+ core sessions
   - Premium users see all 50+ sessions

## Using Sessions

### Starting a Session

1. **Select a Session**
   - Open the Session Library
   - Click on a session card
   - Choose your preferred length (e.g., 20m, 30m, 45m)

2. **Prepare for Session**
   - **Connect Headphones** (required for binaural beats)
   - Find a comfortable position
   - Ensure you won't be disturbed

3. **Start the Session**
   - Click the play button (▶️) in the Audio Controls
   - The session timer will begin counting down
   - Waveform visualization will show live audio data

4. **During the Session**
   - Close your eyes and relax
   - Let the brainwave entrainment guide your state
   - Adjust volume if needed (use the volume slider)
   - The session will automatically stop when the timer reaches zero

## Pro Mode

### Enabling Pro Mode

1. Click the "⚙️ Pro Mode" button in the navigation bar
2. Advanced controls will appear below the waveform

### Pro Mode Features

**Audio Mode Selection**
- Switch between Binaural Beats and Isochronic Tones
- Binaural beats require headphones
- Isochronic tones work with speakers

**Waveform Selection**
- **Sine:** Pure, smooth tone (recommended)
- **Square:** Sharp, digital tone
- **Triangle:** Soft, mellow tone
- **Sawtooth:** Bright, buzzy tone

**Frequency Controls**
- **Carrier Frequency:** Base frequency (0.001 - 9999 Hz)
- **Beat/Pulse Frequency:** Entrainment frequency (0.001 - 9999 Hz)
- Use logarithmic sliders for precise control
- Enter exact values in the number inputs

### Creating Custom Sessions

1. Enable Pro Mode
2. Select your preferred mode (binaural/isochronic)
3. Choose a waveform
4. Set carrier frequency (typically 200 Hz for binaural)
5. Set beat frequency based on desired brainwave state:
   - Delta: 1-4 Hz
   - Theta: 4-8 Hz
   - Alpha: 8-13 Hz
   - Beta: 13-30 Hz
   - Gamma: 30-100 Hz
6. Adjust volume to comfortable level
7. Start the session

## Troubleshooting

### Audio Not Playing

1. **Check Browser Permissions**
   - Some browsers require user interaction to start audio
   - Click the play button to resume audio context

2. **Check Headphones**
   - Ensure headphones are connected
   - Test with system audio settings
   - Try different headphones

3. **Check Browser Console**
   - Open DevTools (F12)
   - Look for error messages
   - Check Web Audio API support

### Session Not Starting

1. **Check Session Selection**
   - Ensure a session is selected
   - Verify session length is chosen

2. **Check Pro Mode**
   - If using Pro Mode, verify frequency settings
   - Ensure values are within valid ranges

3. **Refresh the Page**
   - Sometimes a refresh resolves issues
   - Your session selection will be lost

### Timer Not Working

1. **Check Play State**
   - Timer only runs when playing
   - Ensure play button shows "playing" state

2. **Check Session Length**
   - Verify session length is set
   - Re-select session if needed

### Premium Features Not Unlocking

1. **Check Premium Status**
   - Verify upgrade was successful
   - Check localStorage for premium tier

2. **Refresh the Page**
   - Premium status loads on page load
   - Refresh to update status
